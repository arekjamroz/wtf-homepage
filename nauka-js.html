Elementy języka javascript

Język javascript składa się z następujących elementów:

  • zmienne
  • funkcje
  • literały
  • obiekty
  • wartości
  • operacje porównania

Pliki zawierające kod javascript dołączamy do strony html poprzez dopisanie przed znacznikiem </body> następującego kodu:
<script src="nazwa_pliku.js"> </script>
