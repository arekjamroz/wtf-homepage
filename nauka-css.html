Zasady tworzenia CSSów

Style CSS możemy dołączać do strony www na 3 sposoby

  • Zewnętrzny plik, który dodajemy za pomocą poniższej komendy:
    <link href="style.css" rel="stylesheet" type="text/css">
  • Wewnętzny blok styli zamieszczony w sekcji <head> pomiędzy:
    <style> a </style>
  • Bezpośrednio przy tagu html za pomocą atrybuty style=""

Ze względu na specyficzność najlepszym rozwiązaniem jest używanie zewnętrznego pliku.

Specyficzność w CSS

Specyficzność jest bardzo ważną rzeczą w przy tworzeniu stylów i aby nie mieć z nią problemów dobrze jest nauczyć się prawidłowo ostylowywać stronę www.

Wartości specyficzności:

  • 1,0,0,0,0 - taka wartość jest dla właściwości !important przypisanej do właściwości elementu
  • 0,1,0,0,0 - taka specyficzność jest w przypadku, kiedy mamy do czynienia z właściwością inline
  • 0,0,1,0,0 - taka wartość specyficzności jest dla właściowści przypisanych do id
  • 0,0,0,1,0 - taka wartość specyficzności jest dla właściwości przypisanych do class
  • 0,0,0,0,1 - taka wartość specyficzności jest przypisywana dla właściwości przypisanych do znacznika (np. h1)

Przykładowe atrybuty CSS wraz z opisem

padding - margines wewnętrzny, pełny zapis obejmuje: padding: top right bottom left padding: 10px 5px 20px 12px

display - zmienia sposób wyświetlania elementów, elementy:
display: inline - wyświetlani elementów w lini
display: block - wyświetlani elementów w postaci blokowej
display: inline-block - dodaje do elementów właściwości przeciwne niż domyślne z zachowaniem domyślnych

overflow - zmienia sposób wyświetlania treści w elementach, elementy:
overflow: hidden - ukrywa nadmiar treści
overflow: visible - wartosć domyślna, nadmiar treści wychodzi poza pudełko
overflow: auto - w przypadku zbyt dużej ilości treści dodaje skrola, aby treść występowała wewnątrz pudełka

border - dodaje obramowanie do elementu, przykład:
border: 1px solid red; ({grubość} {typ} {kolor})

margin - margines zewnętrzny, zapis podobny do padding
